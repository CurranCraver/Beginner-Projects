Flex-Box

The flexbox is a more dynamic way to arrange elements on a webpage because it's built to negiotate space automatically instead of manually placing things with fixed widths/positions.

Spacing

Every element is a box made up of the content, padding, border, and margin. Spacing controls how elements breath internally and relate to each other externally.

Wrap-ing

Flex-wrap controls whether flex items stay on the line, wrap onto new lines, or wrap in reverse order when the container gets too small.
